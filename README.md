# Gitian-Setup
This is an easy build script for Gitian Based Coins such a bitcoin.

## How To
- Run sudo ./setup
- Enter Github Link of the coin
- Enter branch name or commit hash
- Profit!

## Tested on:
- Bitcoin

 - Ubuntu 17.10
 - ```sudo apt update```
 - ```sudo apt upgrade -y```
 - ```sudo apt install -y dialog```
